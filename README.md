## 🏗️ Featured Projects

### ☁️ TFAKADNI — Cloud-Native Maternal Health Platform
> Private OpenStack cloud (7 nodes) + Kubernetes cluster + intelligent auto-remediation

- Deployed a full **OpenStack** environment (Keystone, Nova, Neutron/OVN, Glance, Cinder, Swift) on Ubuntu Server 24.04
- Set up a **Kubernetes v1.29** cluster with kubeadm, Flannel, and Containerd; deployed a 3-tier app with StatefulSets, PVC, HPA & Ingress
- Built a **Prometheus + Grafana + Alertmanager** monitoring stack with an auto-remediation pipeline that fixes `CrashLoopBackOff` pods via the Kubernetes API
- Backend: **Angular + Spring Boot + MySQL** secured with JWT/BCrypt in an Agile/Scrum workflow

> End-to-end mobile + web app shipped in 6 weeks and deployed on Azure

- **React.js** admin dashboard for centralized management and analytics
- **Node.js + Express.js** REST API with JWT auth and MongoDB
- Deployed on **Azure Linux VM** with Docker, Nginx reverse proxy & network configuration
